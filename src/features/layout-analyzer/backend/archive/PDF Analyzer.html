Pre-processing tool · v2.4

Know if a PDF is worth vectorizing
before you spend a cycle.

Document Layout Analyzer reads your PDF page by page, measures the balance of vector geometry, raster imagery and selectable text, and tells you which pipeline to send it to — so your geometry engine never wastes a minute on a scan.

How it works

04 Steps · ~2.4s per page
01 · Ingest

Parse the PDF

Page-level extraction of vector paths, raster regions and selectable text.

02 · Measure

Count geometry

Lines, curves, hatched fills — every shape contributes to a normalized score.

03 · Normalize

Score the page

Convert raw counts into page-relative percentages so all formats are comparable.

04 · Classify

Route or reject

One verdict per file: vector-rich → pipeline, otherwise → OCR or NLP service.

Built for the geometry pipeline.

Drop in a PDF the way it leaves a CAD plotter, a scanner, a spec writer or a tender package — the analyzer routes each page to the right downstream service so nothing arrives at the wrong door.
